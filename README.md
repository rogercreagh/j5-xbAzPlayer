# j5-xbAzPlayer and j5-xbAzHead
Two Azuracast radio station "Now Playing" modules for Joomla 5

xbAzHead is intended to be used with the xbWreckers Joomla template in position xbazhead which is the left hand end of the header

xbAzPlayer is intended for use in sidebar module positions and has additional options to display coming next song and previous five songs

Both have options for 
 - subtitle at the top
 - PopOut player control at top right which will open a new small window with a player embedded
 - Display of song title, artist name, and album title with scrolling overflow text
 - Display of album image
 - Display of track duration and progress bar
 
 xbAzPlayer also has optional volume control (not yet implemented)
